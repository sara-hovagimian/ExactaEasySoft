Mayores

En esta ventana podés emitir los mayores de las cuentas del plan de cuentas correspondientes a cualquier año fiscal definido, ya sea que se encuentre abierto o cerrado, activo o inactivo, seleccionándolo bajo el título"Año fiscal". El sistema propone el año fiscal activo.

Es posible emitir el mayor de una cuenta en particular o, mediante una vista, emitir los mayores de un conjunto de cuentas.

Para emitir el mayor de una cuenta, una vez indicado el título del informe, ingresá el código de la cuenta y el rango de fechas de los asientos a incluir. 

Acá podes ver un ejemplo de listado: Mayor por cuenta.

Para realizar la emisión de varias cuentas utilizando una vista, la pantalla se completa en forma similar y sólo tendras que seleccionar la vista a utilizar. 

Si tenes dudas en como definir una vista, podes ver este video.

Acá podes ver un ejemplo de listado de un Mayor en base a una vista.
